Project Overview
By leveraging Power BI's data visualization and exploration capabilities, we were able to gain valuable insights into Zomato's operations, including:
-  Longitude, Latitude: Location coordinates
- Cuisines: Types of food served
- Currency: Local currency (e.g., INR)
- Has_Table_booking, Has_Online_delivery, Is_delivering_now: Services offered
- Switch_to_order_menu: Menu option
- Price_range: Cost category
- Votes, Average_Cost_for_two, Rating: Popularity & pricing
- Datekey_Opening: Opening date
- countryID, country, name: Location details

Key Features
Dynamic and Interactive: Filter, slice, and dice the data to uncover hidden gems and gain a competitive edge.
